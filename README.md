# The-Lounge

### A real-time chat application built using:

- React.js
- Node.js
- MongoDB

### Currently developing features:

- Signin / signup
- Create, delete, update posts
- Feed
- Checkout following users' posts
- Like posts
- Checkout other users
- Follow other users
- Maintain a profile page

### Future plans:

- Real-time chat between 2 users
